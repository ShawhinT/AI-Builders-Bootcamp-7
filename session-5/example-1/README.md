# Fine-tuning GPT-4.1-mini for LinkedIn Posts
In this example, we use OpenAI's fine-tuning API to teach GPT-4.1-mini how to write LinkedIn posts in my style.

## Quick Start

```bash
# Install dependencies
uv sync

# Set up environment
cp .env.example .env
# Add your OpenAI API key to .env

# Run the agent system
uv run jupyter lab
```
